## 概要
修士研究で使用したプログラム群です．

## コンテンツ
- AAPF_RVF
- LVF
- NLMPC
- ObcExperiment
  
### AAPF_RVF
適応仮想ポテンシャル場に回転ベクトル場を組み合わせた誘導のシミュレーションコード．
適応仮想ポテンシャル場のみのシミュレーションもできる．

### LVF
リアプノフベクトル場を使ったシミュレーション．</br>
`LVF_GC_copy.m`を実行する．

### NLMPC
非線形モデル予測制御を使ったシミュレーション

### ObcExperiment
Spresenseに提案手法を実装して実験した．
